This is a lightweight eye tracker written in Python. It automatically detects the pupil and fits an ellipse after applying several different filtering and contour detection algorithms.  

The code is open-sourced and can be found at the following GitHub repository: https://github.com/JEOresearch/EyeTracker/

Hey everyone. This is a lightweight eye 
tracker written in Python that is open source. The green ellipse in the video is 
fit as closely as possible to the pupil, and this is tested on some fairly difficult 
benchmark videos with reflections, eyelashes, and pupil occlusions. The nice thing is that it's 
only around 500 lines of code and can be run with just numpy and openCV. There's quite a bit going 
on in the background that you don't see here, which I plan to go over in another video. 
It's only one file so it should be pretty straightforward to use, but let me know if 
you have any questions in the comments or feel free to post issues directly to the 
GitHub. I'll continue to optimize this, so feel free to subscribe if you want to see the 
improved accuracy over time. Thanks for watching.
