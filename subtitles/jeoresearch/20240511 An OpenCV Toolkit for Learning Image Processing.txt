Title: An OpenCV Toolkit for Learning Image Processing
Date: 20240511
Description:
This toolkit is an interactive OpenCV tutorial that allows you to test different types of image processing. Whether you're a beginner or an advanced user, this tool is designed to help you understand various OpenCV functions. You can use a video from your local machine or connect to your webcam.

I have included features such as Binary Thresholding, Canny Edge Detection, Gaussian Blur, Grayscale Conversion, Hough Line Transform, Contour Detection, Erosion, and Dilation. In addition, the respective parameters for each function are included as sliders, which allow you to change the video in output in real time. You can check the functions you want to activate, and they are applied in the order that you checked them. 

The toolkit is open sourced and can be found at the GitHub link below. Happy testing!! 
https://github.com/JEOresearch/OpenCVToolkit
